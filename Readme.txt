Seach engine - phase 1 
*******************************************************************************************************************************
Tools needed:
1- phpMyAdmin.
2- Wamp server.
*******************************************************************************************************************************
How to run the project:
1- Import the 'database.sql' file into your DBM software (i.e. phpMyAdmin).
2- Move the 'seeds.html' file into your 'www' subfolder in 'wamp' folder or the equivelant folder if you're using another tool.
3- Open Wamp and make sure Apache & mysql services are running.
*2 ways To run the project*:
I-  Run from commandline: Extract searchEngineJar.zip into C drive -> Open commandline -> run this command: java -jar "C:\searchengine.jar"
II- Run from IDE:         Open your java IDE -> open the .java files in your IDE -> add the provided libs -> run 'Crawler.java'
